# Assets Directory

Static assets for the AI Resume Analyzer application.

## Structure

```
assets/
├── images/          ← Static images used in the Streamlit UI
├── architecture/    ← Architecture diagram source files (.drawio, .png)
└── demo/            ← Demo GIFs, screenshots, and videos
```

## Placeholders

Add the following assets to complete the README:
- `images/screenshot_home.png`       — Home page screenshot
- `images/screenshot_results.png`    — Results dashboard screenshot
- `images/screenshot_keywords.png`   — Keyword gap analysis screenshot
- `architecture/architecture.png`    — Architecture diagram export
- `demo/demo.gif`                    — Application demo recording
