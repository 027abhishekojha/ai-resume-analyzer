# 📓 Notebooks — AI Resume Analyzer

Jupyter notebooks for exploratory data analysis, prototyping, and model development.

## Notebook Index

| Notebook | Purpose | Status |
|---|---|---|
| `01_data_exploration.ipynb` | EDA on resume dataset — class distribution, text length stats, vocabulary analysis | 🚧 Planned |
| `02_preprocessing_pipeline.ipynb` | Step-by-step walkthrough of the NLP preprocessing pipeline | 🚧 Planned |
| `03_feature_engineering.ipynb` | TF-IDF vectorization experiments, n-gram analysis, vocabulary inspection | 🚧 Planned |
| `04_model_training.ipynb` | Train and compare multiple classifiers (LR, RF, SVM) | 🚧 Planned |
| `05_model_evaluation.ipynb` | Evaluation metrics, ROC curves, confusion matrix, SHAP analysis | 🚧 Planned |

## Running Notebooks

```bash
# Install Jupyter
pip install jupyter

# Launch Jupyter Lab
jupyter lab notebooks/
```

## Important

> Notebooks are for exploration and prototyping only.
> All production code should live in the `src/` package, not notebooks.
> Notebooks are NOT required to pass CI.
